MathBot

Mathematics for Discord

 

About

MathBot is a discord bot that has a number of features designed to help with maths.

It currently has three main features:

  • Query Wolfram|Alpha
  • Render LaTeX
  • Calculate Equations

Information Commands

=help

Get help from the bot in discord. Help will be sent as a private message.

=help topic

Get help on a particular topic or command. Help will be sent as a private message

The following topics exist:

  • latex
  • wolf
  • calc

=about

Gets some information about the bot.

=stats (private message only)

Returns some statistics from the bot, namely the total number of servers and its current uptime.

Querying Wolfram|Alpha

The =wolf query command is used to query Wolfram|Alpha. query can be anything that you'd be able to put into Wolfram|Alpha normally. Due to both the speed of W|A and the amount of data that gets send to Discord, this command can be slow, so please be patient.

Examples:

  • =wolf intersection of sin(x) and cos(x)
  • =wolf plot x^3 - x^2 + x - 1
  • =wolf dark blue

Rendering LaTeX

The =tex equation command is used render LaTeX equations. query can be any valid LaTeX equation.

Examples:

  • =tex x = 7
  • =tex \sqrt{a^2 + b^2} = c
  • =tex \int_0^{2\pi} \sin{(4\theta)} \mathrm{d}\theta

Notes

The there are a few non-standard shortcuts to make your life easier. For example, * is replaced by \times. Parenthesis ( and ) get \left and \right added to them automatically. In some cases, the bot will recognize malformed LaTeX, and will tell you how to correct it. These features can be disabled by using the the =rtex command instead.

Rendering is done by QuickLatex.com.

Limitations

The bot currently uses an external rendering service. The following features are known to fail:

  • \begin and \end, for any reason, including align and equation.
  • $ to start and end the equations. These are not required, and may do strange things. Use \text{words} to place text.
  • \usepackage, for any reason.
  • Macros such as \@ifpackageloaded.

Other features such as loading images and whatnot will definitely break things

Performing Calculations

The calculator can be invoked with either =calc equation or ==equation.

Examples:

  • =calc 2 ^ (1 + 3) * 5
  • =calc round(10 / 3)
  • =sort 8d6 : 10

Operations

  • + : addition
  • - : subtraction
  • * : multiplication
  • / : division
  • ^ : exponentiation (has some limitations to prevent exploits)

Constants

  • pi : 3.141592...
  • e : 2.178281...

Functions

  • round : rounds a number to the nearest integer
  • int : rounds *down* to the nearest integer
  • sin : sine of an angle (in radians)
  • cos : cosine of an angle (in radians)
  • tan : tan of an angle (in radians)
  • deg : converts radians to degrees
  • rad : converts degrees to radians

Dice Rolling

  • dX : rolls an X-sided die. (e.g. d20)
  • YdX : rolls Y X-sided dice and gives the total. (e.g. 2d6)

You cannot roll more than 1000 dice at once.

Repeating Equations

Appending the equation with : R, where R is an integer will repeat the calculation that many times. This is only usefull only if you want to roll many dice at once. Using the =csort or =sort command will sort the numbers in increasing order. R may not be greater than 50.
